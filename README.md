# Project Name

Meal Tracker App with angular2

## Installation

1. Clone Repo
2. $ npm install
3. $ bower install
4. Install globals
5. $ npm install gulp -g
6. $ npm install bower -g
7. $ gem install sass
8. $ npm install typescript -g
9. $ apm install atom-typescript
